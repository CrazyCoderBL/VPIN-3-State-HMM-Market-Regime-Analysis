# VPIN-3-State-HMM-Market-Regime-Analysis
This project uses a 3-state Hidden Markov Model (HMM) and VPIN (Volume-Synchronized Probability of Informed Trading) to analyse market behaviour and identify whether the market is in a relatively bullish/calm, uncertain/choppy, or high-risk/crisis condition.
The main aim is to study whether being cautious during periods of elevated market toxicity and volatility could have helped preserve returns compared with remaining fully exposed throughout all market conditions.
The model is demonstrated on AAPL daily market data.
